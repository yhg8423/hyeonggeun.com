<h1>
  hyeonggeun.com
</h1>

Terminal design-based personal website of Hyeonggeun, built with [Astro](https://astro.build).

![This is an index page of website](https://github.com/yhg8423/hyeonggeun.com/blob/main/src/images/website.png?raw=true)

You can see the website [here](https://www.hyeonggeun.com)

## Commands

| Command | Action |
|---------|--------|
| `npm install` | Install dependencies |
| `npm run dev` | Start dev server at `localhost:4321` |
| `npm run build` | Build to `./dist/` |
| `npm run preview` | Preview build locally |
